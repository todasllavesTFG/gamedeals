# GameDeals Frontend

Frontend de GameDeals construido con React 19, Vite 8 y Tailwind CSS 4.

## Scripts

```powershell
npm install
npm run dev
```

La aplicación queda disponible en http://localhost:5173.

## Comandos útiles

```powershell
npm run build
npm run lint
npm run preview
```

El frontend espera que la API esté disponible en http://localhost:8000.
